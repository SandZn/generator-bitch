# generator-bitch

A simple generator to scaffolding web apps. Client, Back, or both.
Including: 

- Views
  - Jade
  - EJS
  - Html

- Scripts
  - Javascript
  - CoffeeScript

- Style
  + Sass
  + Less
  + CSS

Tools and frameworks:
- jQuery (2x)
- Angular.js
- Bootstrap
- Foundation

Build System:
- Gulp
- Grunt

and for backend:

- Node.js
- Express 4x
- Hapi.js
- Restify

databases:
- MongoDB
- Redis
- MySQL
